## 🛠️ Corrections de bugs / sécurité

- **Sécurité — CVE-2026-44578 (SSRF Next.js 14)** :

- **Stabilisation des tests d'embeddings** :
  - Correction d'un mock OpenAI muté entre les tests qui faisait boucler le retry d'embedding au-delà du timeout.
