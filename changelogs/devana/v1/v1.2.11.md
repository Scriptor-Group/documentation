## 🛠️ Corrections de bugs / sécurité

- **Fix — TTS bloqué pour les agents partagés** :
  - La lecture audio (Text-to-Speech) d'un message renvoyait une erreur 403 lorsque l'agent était partagé (directement ou via une équipe) sans être public ni possédé par l'utilisateur. Le contrôle d'accès s'appuie désormais sur la même logique que le chat (propriétaire, partage direct, partage par équipe, agent public).

- **Fix — Filtrage par métadonnées SharePoint** :
  - Le filtrage sur les champs de liste SharePoint (ex. type de fichier) rejetait silencieusement tous les fichiers dès qu'un filtre portait sur un champ non expand par défaut dans la réponse de l'API SharePoint.
  - Les opérateurs d'exclusion (« différent de », « ne contient pas ») laissent désormais passer un fichier dont la valeur est absente ou vide, au lieu de le rejeter à tort.
  - Le filtre « vide / non renseigné » reconnaît maintenant aussi bien l'absence de valeur que les chaînes vides.
