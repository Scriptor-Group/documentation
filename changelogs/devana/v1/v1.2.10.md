## ✨ Nouvelles fonctionnalités & améliorations

- **Transmission de la langue détectée aux tools** :
  - La langue détectée de l'utilisateur est désormais envoyée dans un header HTTP `X-language-detect` lors de l'exécution des tools (y compris les tools parents / pre-tools), permettant aux endpoints en aval d'adapter la langue de leur réponse.

- **Refonte de l'endpoint /health/services** :
  - Structure de réponse JSON unifiée et parsable de façon fiable (statuts cohérents, champs stables).
  - Timeouts par service respectés : l'erreur de health check indique désormais quel service est en cause.
  - Une opération vectorielle en cours ne masque plus une vraie panne d'infrastructure.

## 🛠️ Corrections de bugs / sécurité

- **Sécurité — Patch des CVE runtime high/critical** :
  - Correction de 31 alertes Dependabot (sévérité haute/critique, scope runtime) sur les lockfiles serveur et front-end via des bumps ciblés de dépendances (dont axios).

- **Nettoyage des caractères nuls dans les fichiers** :
  - Les caractères nuls (`\0`) présents dans les contenus de fichiers traités par l'API sont désormais remplacés par une chaîne vide (sécurisation des contents).

- **Correction UI — FileDebugPanel** :
  - Correction de l'affichage de la date dans le panneau de debug des fichiers.
