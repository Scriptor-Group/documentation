# Version 1.2.3 → 1.2.4

---

## ✨ Nouvelles fonctionnalités & améliorations

- **Planification de l'heure de synchronisation des sites web** :
  - Il est désormais possible de choisir à quelle heure un site web est automatiquement mis à jour dans votre base de connaissances.
  - Plus de contrôle sur le timing des actualisations de contenu.

- **Alerte sur les agents utilisant un modèle obsolète** :
  - Un message d'avertissement s'affiche désormais lorsqu'un agent utilise un modèle d'IA qui n'est plus supporté.
  - Vous êtes ainsi guidé pour mettre à jour vos agents avant que cela n'impacte leur fonctionnement.

- **Amélioration de la couverture des traductions** :
  - De nombreux écrans de l'interface auparavant partiellement traduits sont désormais entièrement disponibles dans toutes les langues supportées (français, anglais, espagnol, allemand, italien, portugais, arabe, chinois et plus encore).

- **Mises à jour des modèles d'IA disponibles** :
  - La liste des modèles disponibles a été mise à jour pour refléter les dernières versions proposées par OpenAI, Anthropic, Mistral et d'autres fournisseurs.
  - Les modèles obsolètes ne sont plus proposés lors de la création d'un agent ou dans le chat.

---

## 🛠️ Corrections de bugs / sécurité

- **Renforcement de la sécurité sur l'ajout de sources web** :
  - Des contrôles supplémentaires ont été mis en place pour prévenir tout accès non autorisé lors de l'import de contenus depuis des URLs externes.

- **Correction de l'exclusion d'URLs lors du crawl de sites web** :
  - Les règles d'exclusion de pages lors de l'import d'un site fonctionnent désormais correctement quelle que soit la casse des URLs (majuscules/minuscules).

- **Divers correctifs de stabilité** :
  - Plusieurs corrections mineures sur l'affichage et le comportement des modèles dans le chat et la création d'agents.
  - Mises à jour des librairies de stockage cloud pour une meilleure fiabilité.
