# Sous-traitants situés hors de l'Espace Économique Européen

---

## Présentation

Dans le cadre de la gestion du support client via notre plateforme **TyR**, DEVANA fait appel à un nombre limité de sous-traitants situés en dehors de l'Espace Économique Européen (EEE). Cette page détaille ces sous-traitants, la nature des données concernées et les garanties mises en place pour protéger vos données.

Le stockage principal de vos données (base de données TyR) est assuré en **France** (infrastructure Scaleway).

---

## Sous-traitants concernés

### GitHub (Microsoft Corporation)

| | |
|---|---|

| **Rôle** | Gestion et suivi des tickets de support créés dans TyR |
| **Données concernées** | Identifiants de profil (nom, email professionnel), contenu des tickets |
| **Garanties** | EU-US Data Privacy Framework · Clauses Contractuelles Types (CCT) |

### Google Firebase — Cloud Messaging

| | |
|---|---|
| **Siège** | États-Unis |
| **Rôle** | Envoi de notifications push lors de mises à jour de vos tickets TyR |
| **Données concernées** | Identifiants d'appareil (tokens FCM), résumé des notifications de tickets |
| **Garanties** | EU-US Data Privacy Framework · Clauses Contractuelles Types (CCT) |

---

## Encadrement des transferts

Tous les transferts de données vers les États-Unis sont encadrés par :

- **EU-US Data Privacy Framework (DPF)** — Décision d'adéquation de la Commission européenne. GitHub (Microsoft) et Google sont certifiés DPF.
- **Clauses Contractuelles Types (CCT)** — Conformes à la décision d'exécution 2021/914 de la Commission européenne.
- **Mesures techniques** :
  - Chiffrement TLS pour toutes les communications
  - Authentification sécurisée (OAuth 2.0 / OIDC)
  - Aucune donnée sensible (au sens de l'article 9 du RGPD) n'est transférée vers ces sous-traitants

## Vos droits

Vous pouvez à tout moment exercer vos droits d'accès, de rectification, d'effacement, de portabilité ou d'opposition en contactant notre Délégué à la Protection des Données (DPO). DEVANA reste votre interlocuteur unique, y compris pour les données traitées par ses sous-traitants.
